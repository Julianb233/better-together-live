  <name>Task 2: Add cookie relay middleware and legacy cookie cleanup</name>

  <action>
**Add supabaseCookieRelay middleware to middleware.ts:**

Create a new middleware function `supabaseCookieRelay()` that:
1. Calls `await next()` first (let the route handler run)
2. After next(), reads `c.get('supabaseResponseHeaders')` (the Headers object set by createClientWithContext)
3. If headers exist, iterates them with `headers.forEach((value, key) => { c.header(key, value, { append: true }) })` to apply Set-Cookie headers to the actual response

This ensures that when @supabase/ssr refreshes tokens during a request, the new cookies reach the browser.

**Add clearLegacyCookies middleware to middleware.ts:**

Create a middleware function `clearLegacyCookies()` that:
1. Checks if request Cookie header contains `bt_access_token`
2. If found, uses `deleteCookie` from `hono/cookie` to clear `bt_access_token` and `bt_refresh_token` with `{ path: '/' }`
3. Calls `await next()`

Import `deleteCookie` from `hono/cookie` in middleware.ts for this.

**Export both from index.ts barrel:**
Add `supabaseCookieRelay` and `clearLegacyCookies` to the barrel exports.

**Apply in index.tsx:**
- Import `supabaseCookieRelay` and `clearLegacyCookies` from `./lib/supabase`
