<objective>
Add OpenAI as a second AI provider tier and implement intelligent routing between Claude (complex) and OpenAI (simple).

Purpose: Tiered routing reduces costs by sending simple questions to the cheaper OpenAI gpt-4o-mini model while reserving Claude for complex relationship coaching. Fallback logic ensures the coach stays available even if one provider is down.

Output: A coach-router module that classifies questions and selects the appropriate model, with graceful fallback between providers.
</objective>

  <files>src/lib/ai/coach-router.ts</files>
  <action>
    Create `src/lib/ai/coach-router.ts` with:

    1. `classifyQuestion(message: string): 'complex' | 'simple'` function:
       - Complex indicators: words/phrases like 'how do i', 'help me', 'struggling', 'advice', 'what should', 'feeling', 'partner', 'relationship', 'conflict', 'trust', 'intimacy', 'communicate', 'jealous', 'cheating', 'divorce', 'therapy', 'depressed', 'anxious', 'hurt', 'betrayal', 'boundaries'
       - Rules: message > 15 words OR contains complex indicator => 'complex'
       - Otherwise => 'simple'
       - No LLM call for classification (free heuristic)

    2. `getModel(tier: 'complex' | 'simple')` function:
       - complex => `anthropic('claude-haiku-4-5')`
       - simple => `openai('gpt-4o-mini')`
       - Returns the model object for use with generateText

    3. `generateCoachResponse(message, conversationHistory, systemPrompt)` function:
       - Classify question via classifyQuestion
       - Get primary model via getModel
       - Try generateText with primary model
       - On error: log warning, try fallback model (if complex failed, try simple model and vice versa)
       - On both fail: return a friendly static fallback message: "I'm having trouble connecting right now. Here's a general tip: take a moment to breathe and reflect on what you appreciate about your partner. Try again in a few minutes."
       - Return `{ text, tier, model: modelName }` where modelName is the actual model string used (e.g., 'claude-haiku-4-5' or 'gpt-4o-mini')

    Import `generateText` from 'ai', `anthropic` from '@ai-sdk/anthropic', `openai` from '@ai-sdk/openai'.

    Type the conversationHistory param as `Array<{ role: 'user' | 'assistant'; content: string }>`.
