## Campaign Overview

Better Together Live is a relationship intelligence platform offering AI-powered coaching, couple dashboards, communication exercises, and therapist tools. This campaign targets couples 25-45 who want to strengthen their relationship through data-driven insights, plus therapists/coaches who want AI-assisted tools.

---

## Pre-Launch (Weeks -4 to -1)

### Landing Page
- [ ] Deploy landing page with interactive relationship health quiz
- [ ] Hero: "Know Your Relationship Better Than Ever"
- [ ] Funnel: Quiz → results → personalized plan → trial signup
- [ ] Set up Stripe products: `btl_free`, `btl_together`, `btl_thrive`, `btl_coach`
- [ ] Configure 7-day free trial on Together tier
- [ ] Couple billing: one subscription covers both partners

### Relationship Quiz (Lead Magnet)
- [ ] Build 10-question relationship health assessment
- [ ] Personalized results page with actionable insights
- [ ] Email capture at results stage (target 20%+ capture rate)
- [ ] Shareable results card for social media

### Therapist Partnership Outreach
- [ ] Identify 50 target therapists/relationship coaches
- [ ] Create therapist partnership one-pager (PDF)
- [ ] Offer free Coach tier ($79.99/mo value) to 20 pilot therapists
- [ ] Draft therapist outreach email sequence (3 touches)
- [ ] Create "Therapist Integration Guide" for pilot partners

### Content Seeding
- [ ] Start posting relationship tips on Instagram (3x/week)
- [ ] Create 5 TikTok/Reels: "Things couples fight about" + AI resolution demos
- [ ] Draft 3 YouTube scripts: relationship communication hacks
- [ ] Reach out to 5 relationship podcasts for guest spots
- [ ] Create Skool community: "Relationship Intelligence"

### Email Sequences
- [ ] Quiz results follow-up sequence (5 emails): results → deeper insights → exercises → trial offer
- [ ] Welcome sequence (7 emails): couple story → platform value → feature tour → trial
- [ ] Therapist sequence (5 emails): pilot program → client tools → integration → testimonial
- [ ] Weekly relationship tip newsletter
